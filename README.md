A static library project that simplifies NSOperationQueue

## Installation

#### Podfile

```ruby
platform :ios, '8.0'
pod 'CoreOperation', '~> 1.0'
```

Then, run the following command:

```bash
$ pod install
```

#### Old school

Drag into your project the folder `/CoreOperation-iOS`. That's all.
