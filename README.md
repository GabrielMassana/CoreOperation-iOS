# CoreOperation-iOS

A static library project that simplifies NSOperationQueue

# How to Install it

#### Podfile

```ruby
platform :ios, '8.0'
pod 'CoreOperation', '~> 1.0'
```
#### Old school

Drag into your project the folder `/CoreOperation/CoreOperation/CoreOperation`. That's all.
